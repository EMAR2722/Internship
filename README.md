# INFNOVA Academy – Frontend Project

## 📌 Overview
INFNOVA Academy is a modern online learning platform interface built using Next.js.  
This project focuses on building a responsive and user-friendly frontend for browsing courses, viewing detailed course information, and allowing users to enroll in courses.

The platform showcases multiple technology courses such as Web Development, Machine Learning, Cloud Computing, Mobile Development, Cybersecurity, Data Analytics, UI/UX Design, and Blockchain.

---

## 🚀 Features

- Responsive course listing page
- Course search functionality
- Course filtering (Beginner, Intermediate, Advanced)
- Detailed course information page
- Instructor information section
- Enroll and Wishlist buttons
- Modern and clean UI design
- Navigation bar and footer

---

## 🛠️ Technologies Used

- Next.js
- React
- JavaScript
- HTML
- CSS

---

## 📚 Pages Implemented

### 1️⃣ Courses Page
Displays all available courses with:
- Course image
- Course category
- Instructor name
- Course duration
- Number of enrolled students
- Course rating

Users can search and filter courses based on difficulty level.

### 2️⃣ Course Details Page
Provides detailed information about a specific course including:
- Course description
- What students will learn
- Instructor information
- Course duration
- Enrollment section

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/EMAR2722/Internship.git
```

Navigate to the project directory

```
cd INFNOVA
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

Open in browser

```
http://localhost:3000
```
